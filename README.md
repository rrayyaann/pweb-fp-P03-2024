## Install

```bash
npm create vue@latest (install)
npm install lucide-vue-next
npm install swiper
npm add @nuxtjs/tailwindcss @pinia/nuxt
```
## Setup

Make sure to install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
